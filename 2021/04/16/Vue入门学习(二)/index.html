Vue&v-on&v-show&v-if&v-bind&v-for&v-model

v-on

  1. 该指令的作用是：为元素绑定事件
  2. 事件名不需要写on
  3. 指令可以简写为@
  4. 绑定的方法定义在methods属性中
  5. 方法内部通过this关键字可以访问定义在data中的数据

v-on例子

v-show

  1. 该指令的作用是根据真假切换元素的显示状态
  2. 原理是修改元素的display，实现显示隐藏
  3. 指令后面的内容，最终都会解析为布尔值
  4. 值为true元素显示，值为false元素隐藏
  5. 数据改变之后，对应元素的显示状态会同步更新

v-if

  1. 该指令的作用是根据表达式的真假来切换元素的显示状态
  2. 本质是通过操作dom元素来切换显示状态
  3. 表达式的值为true,元素存在于dom中；为false，从dom树中移除
  4. 频繁的切换使用v-show,反之使用v-if,前者的切换消耗小

v-bind

  1. 该指令的作用是：为元素绑定属性
  2. 完整写法是 v-bind:属性名
  3. 简写的话可以直接省略v-bind，只保留 :属性名
  4. 需要动态的增删 class 建议使用对象的方式

v-for

  1. 该指令的作用是：根据数据生成列表结构
  2. 数组经常和v-for结合使用
  3. 语法是(item,index) in 数据(item可以自己取任意字母代表)
  4. item 和index 可以结合其他指令一起使用
  5. 数组长度的更新会同步到页面上，是响应式的

v-on 补充

  1. 事件绑定的方法写成 函数调用 的形式，可以传入自定义参数
  2. 定义方法时需要定义 形参 来接收传入的实参
  3. 事件的后面跟上 .修饰符可以对事件进行限制
  4. .enter 可以限制触发按键回车
  5. 事件修饰符有多种

v-model

  1. 该指令的作用是便捷的设置和获取表单元素的值
  2. 绑定的数据会和表单元素的值相关联
  3. 绑定的数据<—>表单元素的值
